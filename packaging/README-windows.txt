大可客户端（Windows）

一、6002 自动推送版
用途：
- 自动抓取邮箱附件
- 生成净值日报
- 按 14:00 / 16:00 / 16:46 自动运行
- 推送到企业微信机器人

首次使用：
1. 启动 DakeClient6002.exe
2. 先填写邮箱配置与企业微信机器人地址
3. 点击“保存并应用配置”
4. 程序会自动自检并试抓一次（不推送）

使用说明：
- 关闭窗口不会退出，默认最小化到托盘
- 只要程序不退出，就会每天自动运行
- 产品白名单中禁用的产品不会出现在报表中
- 主日报图为 8 列，不生成“昨日对比图”

默认数据目录：
- .\data\6002\nav_report

配置文件：
- .\config\config.json

二、6000 合并工具版
用途：
- 选择多个 Excel 文件
- 调用本地合并服务生成汇总结果
- 将输出文件保存到本地目录

使用说明：
1. 启动 DakeClient6000.exe
2. 点击“选择文件”加入多个 xlsx/xls 文件
3. 点击“开始合并”
4. 在日志区查看进度，在输出目录查看结果

默认输出目录：
- .\data\6000\xlsx_merge\out

配置文件：
- .\config\config.json

三、注意事项
- 两个程序互相独立，可以分别使用
- 如需修改邮箱、Webhook、扫描目录，请编辑配置后重新“保存并应用配置”
- 建议将程序加入开机启动，以保证自动任务持续运行
